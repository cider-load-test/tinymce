What is MODx
 MODx (Modular Extension) is a open source Content Management System that is based on Etomite 0.6. MODx was written by Raymond Irving and was initially designed to be a module update for the open source Etomite Content Management System version 0.6 but will become a complete overhaul of Etomite. MODx is distributed under the GPL license and is now run by a professional team of developers from all over the world. Visit http://www.vertexworks.com/forums for more information.

 The MODx Content Management System provides a powerful framework on which to deploy and secure your website and web applications. For example, it gives you a true system for registered web users and groups that is separate from the administration users. You can grant some web users access to one page and others access to another page. You also now easily duplicate documents, folders (and all their children!), chunks and snippets. Most significantly, though is MODx’s ability to empower you to quickly and easily create and maintain a rich and dynamic website like never before. Please see the Getting Started with MODx tutorial
